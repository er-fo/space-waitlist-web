Installation Guide

Get CADAgent running in Fusion 360 in just a few simple steps.

Windows Installation

1

Download CADAgent

Download the CADAgent zip file:

Download for Windows
2

Extract the zip file

Right-click the downloaded CADAgent-Windows.zip and select Extract All.

3

Copy to AddIns folder

Copy the extracted CADAgent folder to your Fusion 360 AddIns directory:

%APPDATA%\Autodesk\Autodesk Fusion 360\API\AddIns

Press Win+R, paste this path, and press Enter to open the folder.

4

Run in Fusion 360

Restart Fusion 360, then:

  • Go to Tools → Add-Ins
  • Find CADAgent and click Run

macOS Installation

1

Download CADAgent

Download the CADAgent zip file (it will auto-extract):

Download for macOS
2

Copy to AddIns folder

Copy the downloaded CADAgent folder to your Fusion 360 AddIns directory:

~/Library/Application Support/Autodesk/Autodesk Fusion 360/API/AddIns

In Finder, press Cmd+Shift+G, paste this path, and press Enter.

3

Run in Fusion 360

Restart Fusion 360, then:

  • Go to Tools → Add-Ins
  • Find CADAgent and click Run

Troubleshooting

CADAgent doesn't appear in Add-Ins

  • Verify the CADAgent folder is directly in the AddIns directory
  • Restart Fusion 360 completely
