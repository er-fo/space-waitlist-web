Installation Guide

Get CADAgent running in Fusion 360 in minutes

1

Download Complete

Great! You've downloaded the CADAgent ZIP file. Extract it to see the CADAgent-CADAgent folder.

2

Quick Install (Recommended)

The easiest way to install CADAgent is using our automated installers:

macOS: Double-click CADAgent-CADAgent/Install-CADAgent-macOS.sh

Windows: Double-click CADAgent-CADAgent/Install-CADAgent-Windows.bat
Success!

The installer will automatically copy CADAgent to the correct Fusion 360 directory with the proper folder name.

3

Manual Install (If needed)

If the automatic installers don't work, follow these manual installation steps:

Step 1: Navigate to your Fusion 360 AddIns directory:

macOS:
~/Library/Application Support/Autodesk/Autodesk Fusion 360/API/AddIns/

Windows:
C:\Users\[YOUR_USERNAME]\AppData\Roaming\Autodesk\Autodesk Fusion 360\API\AddIns

Step 2: Copy the CADAgent-CADAgent folder from the downloaded ZIP

Step 3: IMPORTANT - Rename the copied folder from CADAgent-CADAgent to just CADAgent
Critical: Folder Name

The folder in your AddIns directory MUST be named exactly CADAgent (not CADAgent-CADAgent). Fusion 360 will not recognize the add-in if the folder name is incorrect. Replace [YOUR_USERNAME] with your actual Windows username.

4

Enable in Fusion 360

Once installed:

1. Open Fusion 360
2. Go to Utilities → Add-Ins
3. Find "CADAgent" in the list
4. Click Run to activate
Installation Complete!

CADAgent is now active in Fusion 360. Start creating 3D models with natural language!
